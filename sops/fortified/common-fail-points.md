# Common Fail Points in FORTIFIED Installations

When inspecting a roof for FORTIFIED compliance, be on the lookout for these common contractor mistakes that will result in a failed evaluation.

## Nailing & Fasteners
- **Nail Spacing:** Nails are too far apart on the roof deck. Code requires a maximum of 6 inches on center.
- **Fastener Type:** Contractors using smooth nails instead of the required ring shank nails for deck attachment.

## Sealing & Taping
- **Improper Sealant:** Using the wrong type of sealant on the seams. Ensure it meets IBHS specifications.
- **Drip Edge Lapping:** Drip edge not properly lapped with the underlayment at the eaves and rakes.

> [!NOTE]
> Always refer to the IBHS official manual for specific regional code requirements as they can vary slightly by county. If you identify a fail point, immediately document it with a photo and inform the roofing contractor so they can rectify it before submitting to IBHS.
